# Specs

Lightweight specs for non-trivial site work.

| Directory | Purpose |
|-----------|---------|
| `active/` | In-flight — read before implementing |
| `archived/` | Done or rejected |

## Template

Create `specs/active/YYYY-MM-DD-short-name.md`:

```markdown
# Title

## Problem
…

## Scope
…

## Acceptance criteria
- [ ] …

## Out of scope
…
```

When done, move to `archived/`.
